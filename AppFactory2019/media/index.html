Adding media

App Factory

When you click the Upload File... button, the dialog will appear. Click Browse... to select a file to upload. It can be an image or a sound file.

Navigate to the place on the hard drive the image or sound files are kept. Select and open the one you want to use.

Adding media

App Factory

The file you selected will appear on the Upload File dialog. If it's the right one, click OK to upload it.

Your file will upload in a few moments, depending on file size and network speed. It will appear in the Media pane and is ready to use in your app.

Files can come from the Media Library on this computer, or you can find images by image searching google.com - you can find free images by clicking:

Tools > Usage Rights > Labelled for Reuse.

Download them to the Media Library first, then upload them in the same way.

App Factory

In the Blocks editor, click on playerChoiceImage (or whatever you called the image for your player choices). The drawer will have a block called

set playerChoiceImage.Picture to ...

Drag that onto the event handler for rock and set it to the text "rock.png" (again, your filename might be different - you can check on the Designer editor in the Media pane).

To play a sound, drag the call rockSound.Play block onto the event handler.

(You may have named your sound something different).

 

You can even add call rockSound.Vibrate and make the phone vibrate when this sound plays!
