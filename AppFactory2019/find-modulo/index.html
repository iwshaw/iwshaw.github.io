Using the mod 3 number loop, let's count the steps we take and what that means.

 

So if we take no steps, that is 0 mod 3 = 0. A draw.

If we take one step forward, 1 mod 3 = 1. You lose.

If we take two steps forward, 2 mod 3 = 2. You win.

 

Three steps forward is 3 mod 3 = 0, a draw again.

If you're at 3 and take a step backwards, you arrive at 2.

 

If you're at 2 and take a step backwards, you arrive at 1.

 

If you're at 1 and take a step backwards, where can you go? You must loop around to 3 again.

 

A step backwards is like taking -1 steps, and when you're at 1 already, -1 steps takes you backwards round the loop to 3!

 

So if -1 mod 3 steps is just one step backwards. It's the same as taking 2 steps forward.

Let's look at how many steps you take each time the player wins, loses, or draws a game, and see if there are any patterns.

 

If we subtract one number from the other, we can find the number of steps.

 

DRAWS

1 draws 1, so 1-1 = 0, 0 mod 3 = 0

2 draws 2, so 2-2 = 0, 0 mod 3 = 0

3 draws 3, so 3-3 = 0, 0 mod 3 = 0

 

In every case of a draw, (Draw - Draw) mod 3 = 0.

BEATS

1 beats 3, so 1-3 = -2, -2 mod 3 = 1

2 beats 1, so 2-1 = 1, 1 mod 3 = 1

3 beats 2, so 3-2 = 1, 1 mod 3 = 1

 

In every case of a 'beat', (Winner – Loser) mod 3 = 1.

 

Two steps back is the same as one step forward in our circular world.

In every case of a 'beaten by', (Loser – Winner) mod 3 = 2.

 

This time, one step back is the same as two steps forward. But modulo corrects this for us, no matter what happens.
