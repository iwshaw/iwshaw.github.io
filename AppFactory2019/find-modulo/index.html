So it’s a circular relationship.                                        

That’s if we go forward in our numbering.

 

What if we go backward?

3 => 2 => 1 => 3 => 2 …

Still a circular relationship.      

                                     

So the whole universe of the game is a closed loop. If we go forward, each member is beaten by the next, and if we go backwards, it beats the next.

Can we define some arithmetic for this loop?

Let’s start at rock(one). We’ve taken zero steps. If we take one step forward, that’s paper(two). Another step and that’s scissors(three). But if we take a third step, we’re back to rock(one).

 

So three is out of bounds for this game universe. It doesn’t make sense to the rock paper scissors world. There can only ever be zero, one or two steps, otherwise you just end up where you started. We should just start over at zero. We’ll call this rule modulo, or mod. Let’s make it ‘mod 3’ so we can remember how big the universe is.

So no steps is 0 mod 3 = 0.

One step, 1 mod 3 = 1.

Two steps, 2 mod 3 = 2.

Three steps, 3 mod 3 = 0 again.

 

Similarly, -1 mod 3 = 2, that’s taking a step backwards from zero to the highest number we know.

Here’s another way of looking at it. You go to the shops to buy some ice cream. The cartons are 3 pounds each.

If you go with 0, 1 or 2 pounds, you can’t buy any. But you still have 0, 1 or 2 pounds left over.

If you go with 3 pounds, you buy one, but have zero pounds change.

If you go with 4 pounds, you buy one, and have one pound left over.

If you go with 5 pounds, the change is 2 pounds.

What if you go with 13 pounds? Well, we know that 12 is a multiple of three. So you’ll have 1 pound change.

Modulo is giving the change, and just forgets about how much ice-cream you bought.

We can use it to count the steps to where we end up, no matter the path to getting there.

If we subtract one number from the other, we can find the number of steps.

 

DRAWS

1 draws 1, so 1-1 = 0, 0 mod 3 = 0

2 draws 2, so 2-2 = 0, 0 mod 3 = 0

3 draws 3, so 3-3 = 0, 0 mod 3 = 0

 

In every case of a draw, (Draw - Draw) mod 3 = 0.

BEATS

1 beats 3, so 1-3 = -2, -2 mod 3 = 1

2 beats 1, so 2-1 = 1, 1 mod 3 = 1

3 beats 2, so 3-2 = 1, 1 mod 3 = 1

 

In every case of a beat, (Winner – Loser) mod 3 = 1.

 

Two steps back is the same as one step forward in our circular world.

This time, one step back is the same as two steps forward. But modulo corrects this for us, no matter what happens.
